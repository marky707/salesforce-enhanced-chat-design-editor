# 05_formal-review/output/

Human-authored formal decision records, per review ID subfolder (e.g. `NS-001/NS-001-formal-decision-round-02.md`). Use the ledger's current round. An AI agent must never author files here—including during a fictional demo. Before continuation, run `python3 tools/review_state.py preflight decision <ID>`; preflight validates but never supplies missing human content.
